# Seedance 2.0 输出格式规范

## 标准输出格式

所有生成的镜头组必须严格遵循以下格式:

```
=== Seedance 2.0 镜头组 · 可直接生成 ===
总时长:[X]s | [比例] | Seedance 2.0

📍 镜头1([时长]s)
画面:[画面描述]
关键词:[关键词列表]

📍 镜头2([时长]s)
画面:[画面描述]
关键词:[关键词列表]

[更多镜头...]

=== 📋 使用方法 ===
1. 在Seedance 2.0中,每个镜头单独生成视频
2. 将生成的[N]段视频导入剪映
3. 按镜头顺序拼接,即可得到完整[X]秒短片

=== 💡 提示 ===
- 比例选择:[具体建议]
- 时长选择:按每个镜头的标注时长设置
- 生成技巧:[针对性建议]

⚠️ Seedance 2.0 技术限制
- 不支持写实真人脸部素材(会被系统拦截)
- 视频参考会消耗更多生成额度
- 输入素材总计≤12个文件
- 单段视频时长:4-15秒
```

## 格式元素详解

### 1. 头部信息

```
=== Seedance 2.0 镜头组 · 可直接生成 ===
总时长:[X]s | [比例] | Seedance 2.0
```

**必填字段:**
- **总时长:** 所有镜头时长之和,单位秒(s)
- **比例:** 16:9 / 9:16 / 1:1 / 4:3

**示例:**
```
总时长:10s | 4:3 | Seedance 2.0
总时长:15s | 16:9 | Seedance 2.0
总时长:8s | 9:16 | Seedance 2.0
```

### 2. 镜头信息块

每个镜头包含3个部分:

```
📍 镜头[N]([时长]s)
画面:[画面描述]
关键词:[关键词列表]
```

#### 2.1 镜头编号和时长

**格式:** `📍 镜头[N]([X]s)`

**规则:**
- N从1开始递增
- 时长可以是整数或小数(如2.5s)
- 时长范围:1-6秒(建议2-5秒)

**示例:**
```
📍 镜头1(2.5s)
📍 镜头2(4s)
📍 镜头3(2.5s)
📍 镜头4(1s)
```

#### 2.2 画面描述

**格式:** `画面:[画面描述]`

**结构:** 主体 + 动作 + 场景 + 光线 + 镜头运动 + 风格

**长度:** 建议30-80字(中文)或50-150词(英文)

**示例:**
```
画面:黑白奶牛猫蹲在木质床头柜上,注视床铺准备跳跃,午后柔光,中景侧推,浅景深,写实毛发
```

**禁止内容:**
- ❌ 时间轴:"0-3秒猫蹲着,3-5秒猫跳"
- ❌ 转场:"淡入淡出"、"溶解"
- ❌ 多场景:"先在客厅,然后切换到卧室"
- ❌ 多动作:"猫跳跃,然后落地,接着转身"

#### 2.3 关键词

**格式:** `关键词:[关键词1, 关键词2, 关键词3, ...]`

**规则:**
- 用逗号分隔
- 可以是中文、英文或混合
- 建议4-8个关键词
- 涵盖主体、动作、风格、氛围

**示例:**
```
关键词:cat, realistic, ready to jump, soft light, push in, shallow depth of field
关键词:猫,写实,跳跃准备,柔光,推镜,浅景深
关键词:transformation, golden particles, cat girl, slow motion, magical
```

### 3. 使用方法块

```
=== 📋 使用方法 ===
1. 在Seedance 2.0中,每个镜头单独生成视频
2. 将生成的[N]段视频导入剪映
3. 按镜头顺序拼接,即可得到完整[X]秒短片
```

**规则:**
- 必须包含这3步说明
- [N]替换为实际镜头数量
- [X]替换为实际总时长

### 4. 提示块

```
=== 💡 提示 ===
- 比例选择:[具体建议]
- 时长选择:按每个镜头的标注时长设置
- 生成技巧:[针对性建议]
```

**比例选择建议示例:**
```
- 比例选择:4:3(推荐)或16:9
- 比例选择:9:16(适合抖音快手竖屏)
- 比例选择:16:9(适合B站YouTube横屏)
- 比例选择:1:1(适合Instagram方图)
```

**生成技巧示例:**
```
- 生成技巧:变身镜头建议开启"创意"模式,其他镜头使用"标准"模式
- 生成技巧:所有镜头使用"标准模式"保证产品还原度
- 生成技巧:镜头2的环绕镜头最重要,可多次生成选最佳
```

### 5. 技术限制块

```
⚠️ Seedance 2.0 技术限制
- 不支持写实真人脸部素材(会被系统拦截)
- 视频参考会消耗更多生成额度
- 输入素材总计≤12个文件
- 单段视频时长:4-15秒
```

**规则:**
- 必须包含这4条标准限制
- 可根据具体情况添加额外提醒

## 特殊格式

### 延长镜头格式

```
=== Seedance 2.0 延长镜头组 ===
基于视频:@视频1([原视频描述])
延长时长:[X]s

📍 延长镜头1([时长]s)
画面:延续@视频1的风格,[新内容描述]
要求:与原视频色调一致,动作连贯流畅,保持主体外观
关键词:[关键词列表]

📍 延长镜头2([时长]s)
画面:延续@视频1的风格,[新内容描述]
要求:与原视频一致
关键词:[关键词列表]

=== 📋 使用方法 ===
1. 在Seedance 2.0中,将@视频1上传作为参考视频
2. 生成延长镜头1和2
3. 在剪映中按顺序拼接:原视频 → 延长镜头1 → 延长镜头2
4. 在原视频和延长镜头1之间不添加转场(保持流畅)
5. 完整时长:原视频时长 + [X]秒

=== 💡 提示 ===
- 比例选择:与原视频保持一致
- 时长选择:按每个镜头的标注时长设置
- 生成技巧:[延长相关建议]

=== 🎬 剪映衔接技巧 ===
- 原视频结尾和延长镜头1开头要动作连贯
- 如果衔接不自然,在剪映中:
  * 调整延长镜头1的速度(0.8x-1.2x)
  * 添加极短的溶解转场(0.2s)
  * 微调色调匹配原视频

⚠️ Seedance 2.0 技术限制
- 视频参考会消耗更多生成额度(约2-3额度/镜头)
- 原视频时长应在2-15秒之间
- 原视频质量越高,延长效果越好
- 不支持写实真人脸部素材
```

### 编辑镜头格式

```
=== Seedance 2.0 编辑镜头 ===
原视频:@视频1

📍 编辑镜头1(原视频时长)
保留:@视频1的[保留内容]
修改:[修改内容]
画面:[详细描述]
关键词:[关键词列表]

=== 📋 使用方法 ===
1. 在Seedance 2.0中,上传@视频1作为参考
2. 生成编辑镜头
3. 根据需要在剪映中替换原视频或作为新版本

=== 💡 提示 ===
- 比例选择:与原视频保持一致
- 时长选择:与原视频相同或略有调整
- 生成技巧:[编辑相关建议]

⚠️ Seedance 2.0 技术限制
- 视频参考会消耗更多生成额度
- 修改幅度越大,与原视频差异越大
- 不支持写实真人脸部素材
```

### 多模态引用格式

当使用图片、视频或音频素材时:

```
📍 镜头1(3s)
画面:参考@图片1的[角色/物品]形象,[动作描述]...
素材:@图片1([用途说明])、@图片2([用途说明])
关键词:[关键词列表]
```

或

```
📍 镜头1(3s)
画面:参考@视频1的运镜方式,[主体描述]...
素材:@视频1(运镜参考)
关键词:[关键词列表]
```

## 格式化建议

### 换行规则

- 每个镜头块之间空1行
- 大标题(===)前后各空1行
- 列表项之间不空行

### 缩进规则

- 不使用缩进
- 所有内容顶格写

### 符号使用

- 镜头编号:使用📍
- 分隔线:使用 `===`
- 列表:使用 `-`
- 强调:使用 `**粗体**`

### 语言一致性

**中文输出:**
```
=== Seedance 2.0 镜头组 · 可直接生成 ===
总时长:10s | 4:3 | Seedance 2.0

📍 镜头1(2.5s)
画面:黑白奶牛猫蹲在木质床头柜上...
关键词:cat, realistic, ready to jump
```

**英文输出:**
```
=== Seedance 2.0 Shot Sequence · Ready to Generate ===
Total Duration: 10s | 4:3 | Seedance 2.0

📍 Shot 1 (2.5s)
Scene: A black and white cow cat crouching on a wooden bedside table...
Keywords: cat, realistic, ready to jump
```

**规则:**
- 中文输入 → 中文输出(关键词可以保留英文)
- 英文输入 → 英文输出
- 不提供双语冗余

## 质量检查

输出前必须检查:

### 格式检查
- [ ] 包含所有必需的格式块(头部、镜头、使用方法、提示、限制)
- [ ] 镜头编号连续(1、2、3、4...)
- [ ] 总时长等于各镜头时长之和
- [ ] 使用了正确的符号(📍、===)

### 内容检查
- [ ] 每个镜头只有一个核心动作
- [ ] 没有时间轴描述
- [ ] 没有转场描述
- [ ] 没有多场景切换
- [ ] 镜头运动明确
- [ ] 画面描述长度适中(30-80字中文)
- [ ] 关键词数量合理(4-8个)

### 逻辑检查
- [ ] 镜头之间有逻辑连贯性
- [ ] 镜头时长分配合理
- [ ] 总时长符合用户需求
- [ ] 比例选择合适
- [ ] 生成技巧针对性强

## 常见错误

### 错误1:时长不匹配

❌ 错误:
```
总时长:10s

镜头1(2s)、镜头2(3s)、镜头3(2s)
实际总计:7s ≠ 10s
```

✅ 正确:
```
总时长:7s

或调整镜头时长使其总和为10s
```

### 错误2:缺少必需块

❌ 错误:只有镜头信息,没有使用方法和提示

✅ 正确:包含所有5个格式块

### 错误3:画面描述包含禁止内容

❌ 错误:
```
画面:0-3秒猫蹲着,3-5秒猫跳跃
```

✅ 正确:
```
画面:猫蹲在床头柜准备跳跃
```

### 错误4:关键词过多或过少

❌ 错误:
```
关键词:cat
关键词:cat, kitten, animal, pet, cute, adorable, fluffy, white, black, spotted, jumping, leaping, bedside, table, bedroom, home, indoor, natural, light, sunlight, afternoon, realistic, photographic, 4k, uhd, detailed
```

✅ 正确:
```
关键词:cat, realistic, ready to jump, soft light, push in, shallow depth of field
```

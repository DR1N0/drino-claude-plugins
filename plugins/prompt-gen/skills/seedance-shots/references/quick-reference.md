# Seedance 2.0 快速参考卡

## 🚀 快速开始检查清单

- [ ] 明确视频想法(一句话概括)
- [ ] 确定总时长(默认8-10秒)
- [ ] 选择风格(写实/动画/二次元/水墨/科幻)
- [ ] 准备参考素材(可选)
- [ ] 确认镜头数量(建议3-4个)

## 📝 单镜头提示词公式

```
主体 + 核心动作 + 场景环境 + 光线 + 镜头运动 + 风格质感
```

**示例:**
黑白猫 + 跃向空中 + 卧室场景 + 午后柔光 + 慢动作环绕 + 写实风格

## 🎥 镜头运动速查

| 中文 | 英文 | 效果 | 适用场景 |
|------|------|------|---------|
| 推近 | push in | 缓慢接近主体 | 强调细节、情绪特写 |
| 拉远 | pull out | 逐渐远离主体 | 展示环境、结束镜头 |
| 横摇 | pan left/right | 左右旋转 | 扫视场景 |
| 环绕 | orbital | 360度旋转 | 全方位展示主体 |
| 跟随 | tracking | 跟随主体移动 | 动作场景 |
| 升降 | crane up/down | 垂直运动 | 展示空间层次 |
| 静止 | static | 不移动 | 稳定画面、对话 |

## ⚠️ 核心原则(铁律)

✅ **一个镜头 = 一段视频**
❌ 不写时间轴(0-3秒、3-7秒)
❌ 不写转场(淡入淡出、切换)
❌ 不写多场景(然后...接着...)

## 🔗 多模态引用语法

```
@图片1 作为首帧
@图片2 作为角色形象参考
@视频1 参考运镜方式
@音频1 用于配乐节奏
```

## 📊 Seedance 2.0 技术限制

| 参数 | 限制 |
|------|------|
| **单段时长** | **4-15秒(最低4秒,不支持更短)** |
| 比例 | 16:9 / 9:16 / 1:1 / 4:3 |
| 图片素材 | ≤9张,<30MB/张 |
| 视频素材 | ≤3个,2-15秒,<50MB |
| 音频素材 | ≤3个,≤15秒,<15MB |
| 总文件数 | ≤12个 |
| **禁止** | 写实真人脸部素材 |
| **重要** | 所有镜头≥4s,后期可在剪映中调速 |

## 📱 使用流程

1. **Seedance 2.0 - 全能参考**入口
2. 输入提示词(复制镜头1的"画面"内容)
3. 如有素材,输入"@"唤起引用
4. 生成视频并下载
5. 重复2-4,完成所有镜头
6. **剪映**中按顺序拼接
7. 导出成片

## 🔥 高级技巧

**角色一致性:** 为每个镜头添加"参考@图片1的角色形象"

**运镜复刻:** "完全参考@视频1的所有运镜效果"

**音乐卡点:** 标注"配合音乐重拍"并提供@音频1

**视频延长:** "延续@视频1的风格,[新内容]"

## ❌ 常见错误示例

**错误1:包含时间轴**
```
❌ 0-3秒猫蹲着,3-5秒猫跳跃,5-7秒猫落地
✅ 猫从床头柜跃向空中,慢动作
```

**错误2:多场景切换**
```
❌ 先在客厅跳舞,然后切换到阳台,最后到卧室
✅ 女孩在客厅跳舞,镜头缓慢推近
```

**错误3:复杂转场**
```
❌ 淡入淡出效果,从A场景溶解到B场景
✅ (转场在剪映中处理,单镜头不需要写)
```

**错误4:逐帧控制**
```
❌ 先抬起手,然后转头,接着微笑,最后眨眼
✅ 女孩抬手微笑,柔和面光,静止镜头
```

**错误5:剧烈形变**
```
❌ 猫突然爆炸变成猫娘
✅ 猫被金色光粒子环绕,柔和渐变成猫娘
```

## 🎬 比例选择建议

| 比例 | 适用场景 | 特点 |
|------|---------|------|
| 16:9 | 横屏视频,电影感 | 宽银幕,适合风景、叙事 |
| 9:16 | 抖音、快手竖屏 | 手机原生,适合人物、产品 |
| 1:1 | Instagram方图 | 正方形,适合产品展示 |
| 4:3 | 复古风格,艺术感 | 经典比例,画面饱满 |

## 🌟 推荐工作流

1. **构思阶段:** 一句话描述视频想法
2. **规划阶段:** 确定风格、时长、素材
3. **拆分阶段:** 自动拆分3-4个镜头
4. **生成阶段:** Seedance逐个生成
5. **拼接阶段:** 剪映导入拼接
6. **润色阶段:** 添加转场、配乐、字幕
7. **导出阶段:** 输出成片

---

**记住:** 简单、独立、可生成!每个镜头都应该是可以单独生成的完整视频片段。

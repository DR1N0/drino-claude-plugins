# Seedance 2.0 技术规格详解

## 平台概述

Seedance 2.0是字节跳动即梦AI推出的视频生成模型,核心特点是**单镜头生成架构**。

**关键特性:**
- 一个提示词生成一段完整视频
- 不支持时间轴控制(不能指定"0-3秒做A,3-5秒做B")
- 不支持场景切换(不能在单次生成中包含多个场景)
- 需要用户在剪映等后期工具中拼接多段视频

## 输入限制

### 1. 文本提示词

**长度限制:**
- 建议:50-200字(中文)或100-400词(英文)
- 过短:缺少细节,生成质量不稳定
- 过长:可能被截断,影响理解

**语言支持:**
- 中文(推荐):模型针对中文优化
- 英文:支持,但效果可能不如中文

**提示词结构:**
```
主体 + 动作 + 场景 + 光线 + 镜头运动 + 风格质感
```

### 2. 图片素材

**数量限制:** ≤9张

**单张限制:**
- 大小:<30MB
- 格式:JPEG、PNG、WebP、BMP、TIFF、GIF
- 分辨率:建议512-2048px

**用途:**
- 首帧参考:精确控制起始画面
- 角色形象:保持角色一致性
- 场景参考:复刻环境氛围
- 风格参考:匹配艺术风格

**引用语法:**
```
@图片1 作为首帧
@图片2 作为角色形象参考
参考@图片3的场景氛围
```

### 3. 视频素材

**数量限制:** ≤3个

**单个限制:**
- 时长:2-15秒
- 大小:<50MB
- 格式:MP4、MOV
- 分辨率:建议720p-1080p

**用途:**
- 运镜复刻:复制复杂镜头语言
- 动作参考:匹配动作节奏
- 视频延长:延续已有视频
- 视频编辑:角色替换、剧情颠覆

**引用语法:**
```
@视频1 参考运镜方式
延续@视频1的风格
完全参考@视频1的所有运镜效果
```

**注意事项:**
- 视频参考会消耗更多生成额度
- 素材质量直接影响生成效果
- 避免使用低分辨率或模糊素材

### 4. 音频素材

**数量限制:** ≤3个

**单个限制:**
- 时长:≤15秒
- 大小:<15MB
- 格式:MP3、WAV

**用途:**
- 配乐节奏:音乐卡点生成
- 氛围匹配:配合音乐情绪

**引用语法:**
```
@音频1 用于配乐
配合音乐重拍
```

### 5. 总体限制

**文件总数:** ≤12个(图片+视频+音频)

**组合示例:**
- 9图 + 3视频 + 0音频 = 12个 ✅
- 6图 + 3视频 + 3音频 = 12个 ✅
- 5图 + 2视频 + 2音频 = 9个 ✅

## 输出规格

### 1. 时长

**范围:** 4-15秒/段

**推荐时长分配:**
- 静态镜头:2-3秒
- 动作镜头:3-5秒
- 变身特效:4-6秒
- 对话镜头:3-4秒

**注意:**
- 单段视频不能超过15秒
- 如需更长视频,拆分多个镜头后拼接

### 2. 分辨率

**标准版:**
- 最高:1080p(1920×1080)
- 帧率:24fps

**Pro版(付费):**
- 最高:2K(2560×1440)
- 帧率:24fps/30fps可选

**推荐设置:**
- 社交媒体:1080p 24fps
- 专业用途:2K 30fps(Pro)

### 3. 宽高比

**支持比例:**
- 16:9 - 横屏视频,电影感
- 9:16 - 竖屏视频,抖音快手
- 1:1 - 方图,Instagram
- 4:3 - 复古比例,艺术感

**选择建议:**
| 用途 | 推荐比例 | 原因 |
|------|---------|------|
| 抖音快手 | 9:16 | 竖屏原生体验 |
| B站YouTube | 16:9 | 横屏标准 |
| 小红书Instagram | 1:1或9:16 | 平台适配 |
| 艺术创作 | 4:3 | 画面饱满,经典 |

### 4. 输出格式

**文件格式:** MP4

**编码:**
- 视频:H.264
- 音频:AAC(如有配乐)

**比特率:**
- 视频:约8-12Mbps(1080p)
- 音频:128-192kbps

## 内容限制

### 禁止内容

1. **写实真人脸部素材**
   - 系统会自动拦截
   - 包括:明星脸、普通人脸、儿童脸
   - 允许:卡通脸、二次元、艺术化处理

2. **违规内容**
   - 政治敏感
   - 暴力血腥
   - 色情低俗
   - 侵权内容

### 审核机制

**生成前审核:**
- 提示词敏感词检测
- 素材内容审核(人脸识别等)

**生成后审核:**
- 视频内容审核
- 违规视频会被拒绝生成或删除

## 生成模式

### 标准模式

**特点:**
- 平衡质量和速度
- 适合大多数场景
- 生成速度:约2-4分钟/段

**适用场景:**
- 静态镜头
- 简单动作
- 常规场景

### 创意模式

**特点:**
- 更强的想象力和艺术性
- 适合复杂特效和创意场景
- 生成速度:约3-5分钟/段

**适用场景:**
- 变身特效
- 魔法效果
- 抽象概念
- 艺术风格化

### 选择建议

| 镜头类型 | 推荐模式 | 原因 |
|---------|---------|------|
| 产品展示 | 标准 | 稳定可控 |
| 人物对话 | 标准 | 写实自然 |
| 变身特效 | 创意 | 效果炫酷 |
| 魔法场景 | 创意 | 想象力强 |
| 风景延时 | 标准 | 真实感强 |

## 额度消耗

### 基础消耗

**纯文本生成:**
- 标准模式:1额度/段
- 创意模式:1.5额度/段

**图片参考(≤3张):**
- 标准模式:1.5额度/段
- 创意模式:2额度/段

**视频参考:**
- 标准模式:2-3额度/段
- 创意模式:3-4额度/段

**多模态组合:**
- 图片+视频:3-5额度/段
- 根据素材数量和复杂度动态计算

### 额度获取

**免费用户:**
- 每日:3-5额度
- 签到奖励:1-2额度

**Pro会员:**
- 每月:100-200额度
- 高级功能:2K输出、优先队列

## 最佳实践

### 1. 提示词撰写

**推荐结构:**
```
[主体描述] + [核心动作] + [场景环境] + [光线氛围] + [镜头运动] + [风格质感]
```

**示例:**
```
黑白奶牛猫蹲在木质床头柜上,注视床铺准备跳跃,午后柔光,中景侧推,浅景深,写实毛发
```

**技巧:**
- ✅ 使用具体名词:不写"动物"写"猫"
- ✅ 明确动作方向:不写"移动"写"向右移动"
- ✅ 指定光线类型:不写"光"写"柔光"或"硬光"
- ✅ 强调质感细节:"写实毛发"、"金属光泽"

### 2. 素材准备

**图片素材:**
- 清晰度:至少720p
- 主体明确:避免杂乱背景
- 光线良好:避免过暗或过曝
- 格式统一:建议都用JPEG或PNG

**视频素材:**
- 稳定性:避免抖动严重的素材
- 时长适中:建议5-10秒
- 质量优先:宁可少用也不用低质素材

### 3. 镜头拆分策略

**原则:**
- 一个镜头一个核心动作
- 每个镜头2-4秒
- 避免在单镜头内切换场景
- 使用柔和过渡而非剧烈形变

**拆分模板:**

**变身特效(4镜头):**
1. 准备动作(2.5s)
2. 变身过程(4s)
3. 变身完成(2.5s)
4. 特写展示(1s)

**产品展示(3镜头):**
1. 细节特写(3s)
2. 环绕展示(4s)
3. 品牌收尾(3s)

**情感叙事(4镜头):**
1. 环境建立(2s)
2. 人物动作(3s)
3. 情绪特写(3s)
4. 结局画面(2s)

### 4. 后期拼接

**剪映工作流:**
1. 导入所有生成的视频片段
2. 按镜头顺序排列
3. 添加转场(淡入淡出、溶解等)
4. 调整色调统一性
5. 添加配乐和音效
6. 添加字幕(如需要)
7. 导出成片

**转场建议:**
- 同场景镜头:剪切或快切
- 不同场景:淡入淡出或溶解
- 节奏快的视频:快切
- 情感叙事:柔和转场

### 5. 常见问题解决

**问题1:人物崩坏**
- 原因:提示词包含时间轴或多场景
- 解决:拆分成单一场景单一动作

**问题2:动作不连贯**
- 原因:单镜头内描述了多个动作
- 解决:每个镜头只写一个核心动作

**问题3:画面混乱**
- 原因:提示词过于复杂或矛盾
- 解决:简化描述,保持逻辑一致

**问题4:风格不统一**
- 原因:每个镜头的风格描述不同
- 解决:为所有镜头添加统一的风格标签

**问题5:生成失败**
- 原因:素材违规或提示词敏感
- 解决:检查素材合规性,修改提示词

## 技术限制总结

| 类别 | 限制 | 说明 |
|------|------|------|
| 单段时长 | 4-15秒 | 不可超过 |
| 图片素材 | ≤9张,<30MB/张 | JPEG/PNG/WebP等 |
| 视频素材 | ≤3个,2-15s,<50MB | MP4/MOV |
| 音频素材 | ≤3个,≤15s,<15MB | MP3/WAV |
| 总文件数 | ≤12个 | 图+视频+音频 |
| 真人脸部 | 禁止 | 系统拦截 |
| 分辨率 | 最高1080p(标准) | Pro支持2K |
| 比例 | 16:9/9:16/1:1/4:3 | 可选 |
| 帧率 | 24fps(标准) | Pro支持30fps |
